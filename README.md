# Beach Ball Bounce

This is the repository for the code of "Brach Ball Bounce" a game featured on the [Comfy Jam: Summer 2026](https://itch.io/jam/comfy-jam-summer-2026) game jam. This game was made using Phaser.JS v4 and Packaged with Electron JS as a Windows executable.
